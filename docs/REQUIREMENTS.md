# Recursive Agent — submission requirements

The following text mirrors the official OpenClaw Agenthon 2026 submission requirements. Use it as a checklist while building **Recursive Agent**.

### What to Build
Participants are completely free to use any framework, AI model, API, programming language, or tools. However, the submitted project must include an AI Agent or Multi-Agent System component capable of executing reasoning, automation, decision-making, tool usage, workflow execution, or autonomous interaction.

Note: AI coding assistants (e.g., OpenClaw, Cursor) are allowed. Publicly available open-source templates or libraries are also permitted.

### What to Submit
**Team naming format:** `OpenClaw2026_TeamName`

All submissions must be submitted through Devpost before the deadline. Your submission must include:

- **Project description:** A narrative explanation of the project you have built (Recursive Agent).
- **GitHub repository:** A link to your GitHub repository. The repository must be set to **Public** and include a `README.md` file with clear installation and setup instructions to ensure reproducibility for the judges. (Repository naming format: `OpenClaw2026_TeamName_ProjectName`).
- **Demo video:** A link to a maximum 2-minute demonstration video uploaded to YouTube, Google Drive, or a similar platform. The access mode must be set to **Public** (viewable). The video must clearly showcase the product's workflow, AI Agent capabilities, audio, and visual interface.
- **Pitch deck:** A presentation document (maximum 5 slides) detailing your Problem Statement, Solution Overview, AI Agent Architecture, Tech Stack, and Future Impact. (Must be in PDF format.) Naming format: `OpenClaw2026_TeamName_ProjectName.pdf`.
- **AI tools / models used:** A detailed list of the AI models and supporting tools implemented in your project.
- **Live deployment link (optional):** A link to a live or deployed application/agent. (Projects with a functional live deployment will receive additional consideration from the judges.)
